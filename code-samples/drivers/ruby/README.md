and

[Gitana Platform - Ruby Driver](https://gitana.io/documentation/gitana/4.0/developers/drivers/ruby.html)

# Usage

    1. cd sample
    2. create your gitana.json file in this folder
    3. run 'bundle install'
    4. run 'ruby sample.rb'
